# SYSCCOM Integradores - E-commerce

Tienda en línea de equipos de telecomunicaciones y tecnología.

## Tecnologías

- **Next.js 16** con App Router y TypeScript
- **Tailwind CSS 4** para estilos
- **Zustand** para estado global (carrito)
- **Lucide React** para iconografía

## Características

- Catálogo de productos con filtros por categoría, marca y búsqueda
- Carrito de compras persistente
- Flujo de checkout completo
- Diseño responsive y mobile-first
- Integración preparada para APIs de proveedores (Syscom, Abasteo)
- Página de servicios y contacto

## Categorías de Productos

- Cámaras de Vigilancia (CCTV)
- Redes y Conectividad
- Cableado Estructurado
- Antenas y Enlaces Inalámbricos
- Servidores y Almacenamiento
- Telefonía IP
- Control de Acceso
- Alarmas y Sensores

## Desarrollo

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm start
```
